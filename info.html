    Тег <style> применяется для определения стилей элементов веб-страницы. Тег style необходимо использовать внутри контейнера head. Можно задавать более чем один тег style.

    Тег <img> предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG. Адрес файла с картинкой задаётся через атрибут src. Если необходимо, то рисунок можно сделать ссылкой на другой файл.

    Тег <color> Определяет цвет текста элемента.

    Тег <nav> задает навигацию по сайту. Если на странице несколько блоков ссылок, то в nav обычно помещают приоритетные ссылки. Также допустимо использовать несколько тегов nav в документе.

    Тег<а> определяет гиперссылку

    Тег <h1-h6> определяет заголовки HTML

    Тег <u> oпределяет текст, который должен быть стилистически отличается от обычного текста

    Тег <button> определяет нажатую кнопку
